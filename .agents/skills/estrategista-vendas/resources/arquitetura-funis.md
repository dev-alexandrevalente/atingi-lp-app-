# 🔁 Módulo 3: Arquitetura de Funis

## Modelos de Funil

### 1. Funil Tripwire (Escada de Valor)
```
Ad → Lead Magnet (grátis) → Tripwire (R$7-27) → Core (R$97-297) → Upsell (R$497+)
```
**Quando usar**: Público frio, low ticket como porta de entrada
**Chave**: O tripwire NÃO precisa dar lucro — ele cria compradores

### 2. Funil Webinar
```
Ad → Registro → Webinar ao vivo/gravado → Oferta → Follow-up (e-mail/WhatsApp) → Downsell
```
**Quando usar**: Info-produtos R$497+, público semi-quente
**Chave**: 60% conteúdo / 40% pitch. Ensinar o "O QUÊ" sem o "COMO"

### 3. Funil VSL (Video Sales Letter)
```
Ad → VSL Page → Checkout → Order Bump → Upsell → Downsell
```
**Quando usar**: Produto digital, preço médio (R$47-297)
**Chave**: VSL precisa >30% retention no minuto da oferta

### 4. Funil Challenge (Desafio)
```
Ad → Registro → Desafio 5-7 dias → Oferta no último dia → Community
```
**Quando usar**: Engajamento primeiro, venda depois
**Chave**: Resultados rápidos no desafio = credulidade para o produto

### 5. Funil Gamificado
```
Ad → Quiz/Jogo interativo → Resultado personalizado → VSL → Oferta customizada
```
**Quando usar**: Público frio, diferenciação, coleta de dados
**Chave**: Efeito IKEA — investimento pessoal aumenta conversão

### 6. Funil Application
```
Ad → Conteúdo → Aplicação (formulário) → Call de vendas → High-ticket
```
**Quando usar**: Mentoria, consultoria, R$5k+
**Chave**: A aplicação QUALIFICA e cria comprometimento

### 7. Funil Invisible
```
Ad → Conteúdo gratuito massivo → Free+Shipping → Backend → Subscription
```
**Quando usar**: Livros, produtos físicos como porta de entrada
**Chave**: Lucro está no backend, não no front

## Engenharia de Conversão — 20 Pontos de Otimização

### Na Página
1. **Headline acima do fold**: Teste A/B — impacta 80% da conversão
2. **CTA visível sem scroll**: Botão primário always visible
3. **Velocidade de loading**: <3s (cada 1s = -7% conversão)
4. **Mobile-first**: 80%+ do tráfego Meta é mobile
5. **1 CTA por página**: Sem distrações, sem links de saída
6. **Prova social above the fold**: "X pessoas já..."
7. **Elemento visual dominante**: Imagem/vídeo que comunica o resultado

### No Funil
8. **Micro-compromissos**: Pedir informações progressivamente (nome → email → telefone)
9. **Exit Intent**: Pop-up com oferta alternativa ou desconto
10. **Timer de urgência**: Real e sincronizado (não fake)
11. **Multi-step checkout**: Pedir dados em etapas (vs formulário gigante)
12. **Order bump**: Oferta complementar no checkout (+15-30% AOV)
13. **Retargeting em camadas**: Copy diferente para cada etapa abandonada
14. **Sequência de e-mail/WhatsApp**: 5-7 touchpoints pós-lead
15. **Chatbot de recuperação**: Perguntar por que não comprou

### No Criativo
16. **3 formatos**: Video (VSL curta), Carousel, Imagem estática → testar
17. **Hook nos primeiros 3s**: Se não prender, scroll
18. **Legendas**: 85% assiste sem som
19. **Native feel**: Parecer conteúdo, não anúncio
20. **UGC (User Generated Content)**: Depoimentos em formato casual

## Diagnóstico de Funil (Análise de Gargalo)

### Passo 1: Mapear o fluxo completo
```
Impressão → Clique → Page View → Lead → Engajamento → Conversão → Upsell
```

### Passo 2: Medir conversão entre cada etapa
- Onde o MAIOR dropoff está? → Esse é o gargalo

### Passo 3: Priorizar pelo impacto
- Otimizar o gargalo com MAIOR volume primeiro
- Um funil com 10.000 cliques e 1% de conversão no checkout = 100 vendas
- Melhorar checkout de 1% → 2% = dobrar vendas com ZERO custo extra

### Passo 4: Testar uma variável por vez
- Nunca mudar tudo junto
- A/B test com significância estatística (mínimo 100 conversões por variante)
