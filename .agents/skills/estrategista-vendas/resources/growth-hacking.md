### AARRR — Pirate Metrics
```
ACQUISITION → Como atraímos leads? (análise por canal)
ACTIVATION → O lead engajou? (completou X% do funil)
RETENTION → O lead voltou? (e-mail, remarketing)
REVENUE → O lead comprou? (conversão)
REFERRAL → O comprador trouxe outros? (indicação)
```

### ICE Score — Priorização de Experimentos
```
IMPACTO:    1-10 (quanto pode impactar no resultado)
CONFIANÇA:  1-10 (quão confiante estou que vai funcionar)
FACILIDADE: 1-10 (quão fácil é implementar)

SCORE = (I + C + F) / 3

> 8.0 = FAZER AGORA
5-8   = Próximo sprint
< 5   = Backlog
```

### North Star Metric
**Uma única métrica que define sucesso**

| Tipo de Negócio | North Star | Por Quê |
|-----------------|------------|---------|
| Funil de vendas | Checkout Rate | Mede a eficácia do funil inteiro |
| SaaS | Monthly Active Users | Uso = retenção = receita |
| E-commerce | Revenue per visitor | Volume × conversão |
| Infoproduto | LTV / CAC ratio | Sustentabilidade |

## Táticas de Aquisição Agressiva

### Meta Ads — Estrutura de Campanhas
```
CAMPANHA (Objetivo: Conversão)
├── Ad Set 1: Lookalike 1% (Compradores)
│   ├── Criativo A: Video UGC
│   ├── Criativo B: Carrossel
│   └── Criativo C: Imagem estática
├── Ad Set 2: Lookalike 2% (Leads quentes)
│   ├── Criativo A
│   ├── Criativo B
│   └── Criativo C
├── Ad Set 3: Interesse (peso, saúde, etc.)
│   ├── Criativo A
│   ├── Criativo B
│   └── Criativo C
└── Ad Set 4: Retargeting (visitou, não comprou)
    ├── Criativo D: Depoimento
    ├── Criativo E: Urgência
    └── Criativo F: FAQ/Objeção
```

### Retargeting em Camadas
| Audiência | Tempo | Copy | Oferta |
|-----------|-------|------|--------|
| Visitou página | 0-3 dias | "Esqueceu de algo?" | Mesma |
| Viu 50% VSL | 0-7 dias | Depoimento + resultado | Mesma |
| Iniciou checkout | 0-3 dias | "Sua vaga ainda está reservada" | + Bônus |
| Abandonou checkout | 3-7 dias | FAQ + garantia | Mesma |
| Comprou | 7-30 dias | Upsell/Cross-sell | Premium |

### Loop Viral — K-Factor
```
K = (Convites enviados por usuário) × (Taxa de conversão do convite)

K > 1 = Crescimento exponencial
K = 1 = Estável
K < 1 = Precisa de aquisição paga

COMO AUMENTAR K:
→ Incentivo: "Indique e ganhe 30% no próximo"
→ Social proof: "Compartilhe seu resultado"
→ Conteúdo compartilhável: Quiz com resultado personalizável
→ Exclusividade: "Convide 3 amigas para liberar bônus X"
```
