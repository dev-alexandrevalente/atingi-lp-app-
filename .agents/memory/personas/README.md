# 👤 Personas — Memory System

## Papel
Armazena personas validadas por projeto. Cada persona é um perfil comportamental profundo que pode ser reutilizado entre conversas e skills.

## Quem Escreve
- `product-intelligence` (após framework de descoberta)
- `estrategista-vendas` (após análise de avatar)

## Quem Lê
- `copywriting-engine` (tom de voz, linguagem)
- `funnel-architect` (jornada e níveis de consciência)
- `offer-architect` (dores e objeções)
- `creative-director` (ângulos e hooks)
- `vsl-architect` (narrativa e emoção)
- `recovery-architect` (canais e timing)

## Formato de Arquivo
```
persona_[nome-fictício]_[produto].md
```

## Estrutura Obrigatória
```markdown
# Persona: [Nome Fictício]
- **Produto**: [nome do produto]
- **Criada em**: [data]
- **Validada por**: [skill que gerou]

## Quem É
[contexto, momento, nível de consciência]

## O Que Diz
[frases reais, buscas, reclamações]

## O Que Sente
[frustração, medo, desejo, vergonha]

## O Que Já Tentou
[soluções anteriores e por que falharam]

## Transformação (Antes → Depois)
[tabela antes/depois]

## Objeções Top 5
[lista com raiz emocional e resposta]
```
