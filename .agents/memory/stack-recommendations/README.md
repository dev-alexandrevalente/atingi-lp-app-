# 🛠️ Stack Recommendations — Memory System

## Papel
Recomendações de stack tecnológico validadas por projeto. Ferramentas, serviços, integrações e configurações que funcionaram.

## Quem Escreve
- `arquitetura` (decisões de stack)
- `tracking-implementation` (ferramentas de analytics)
- `funnel-architect` (ferramentas de funil)
- `design-system` (bibliotecas e frameworks CSS)

## Quem Lê
- Skills técnicas ao iniciar projeto novo
- `/orquestrador` para contexto de decisão
