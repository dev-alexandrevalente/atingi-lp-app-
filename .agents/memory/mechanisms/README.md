# ⚙️ Mechanisms — Memory System

## Papel
Mecanismos únicos criados e validados para cada produto. O mecanismo é a explicação diferenciadora de "como" o produto entrega o resultado de forma única.

## Quem Escreve
- `product-intelligence` (ao definir mecanismo único)
- `offer-architect` (ao refinar oferta)

## Quem Lê
- `copywriting-engine` (para copy de mecanismo)
- `vsl-architect` (para narrativa de VSL)
- `creative-director` (para ângulos criativos)
- `funnel-architect` (para estrutura de prova)

## Formato de Arquivo
```
mechanism_[nome-do-produto].md
```

## Estrutura Obrigatória
```markdown
# Mecanismo: [Nome do Mecanismo]
- **Produto**: [nome]
- **Criado em**: [data]
- **Versão**: [N]

## Nome do Mecanismo
[nome proprietário, se houver]

## Explicação em 1 Frase
[como funciona em linguagem simples]

## Por Que É Diferente
[o que separa dos concorrentes/alternativas]

## Prova de Funcionamento
[evidências, estudos, resultados que sustentam]

## Narrativa para Copy
[como contar a história do mecanismo ao lead]

## Objeções que o Mecanismo Quebra
[lista de objeções que ele responde naturalmente]
```
