# Pattern: Lollapalooza Orchestration (Empilhamento de Vieses)
- **Domínio**: copy | funil | VSL | oferta
- **Validado em**: 6+ fontes do corpus doutrinário
- **Confiança**: alta

## Descrição
Parar de usar gatilhos, vieses e mecanismos psicológicos isolados e começar a empilhá-los estrategicamente na mesma direção. 3 blocos de empilhamento que, quando executados em sequência, criam conversão extrema.

## Quando Usar
- Em peças completas: VSLs, LPs longas, funis de venda
- Quando o objetivo é conversão máxima (não awareness)
- Quando todas as informações do lead (persona, Caverna, Inimigo) estão disponíveis

## Estrutura
**Bloco A — Conexão Rápida (Primeiros 20%):**
- Pacing + Barnum + Tribalismo
- Resultado: Barreiras racionais caem completamente

**Bloco B — Urgência Real (20-60%):**
- Choque de Realidade + FOMO Narrativo + Prova Social
- Resultado: Tempestade emocional com base lógica

**Bloco C — Oferta Irresistível (60-100%):**
- Mecanismo Único (Caixa Preta) + Pertencimento + Future Pacing
- Resultado: Preço vira detalhe, conversão máxima

## Exemplos
- VSL Ho'oponopono: Bloco A (validar dor emocional → transferir culpa para indústria de dietas) → Bloco B (revelar ciência do eixo emocional-metabólico → FOMO com vagas limitadas) → Bloco C (protocolo exclusivo → comunidade de fundadoras → visualização de vida sem ansiedade)
- LP SaaS: Bloco A (Barnum com métricas erradas + culpa no método manual) → Bloco B (IA mudando o jogo + clientes similares lucrando) → Bloco C (plataforma exclusiva + trial + pertencimento ao ecossistema)

## Contra-indicações
- Peças curtas (< 30s) → Use no máximo 2 mecanismos empilhados
- E-mails simples → Use Frentista Consultivo puro
- Retargeting quente → Pule Bloco A, vá direto ao B ou C
