# 📋 Product Briefs — Memory System

## Papel
Briefings estratégicos consolidados por produto. É o documento-mestre que todos os agents consultam antes de qualquer tarefa.

## Quem Escreve
- `product-intelligence` (geração obrigatória como output final)

## Quem Lê
- **Todas as skills** — este é o insumo universal

## Formato de Arquivo
```

```
